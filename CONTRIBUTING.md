# Contributing

PM Playbook by Swapnil is a lightweight, practical knowledge base for product managers: guides, templates, checklists, and resources you can apply in real work.

## Repo structure

- Content lives in `src/content/docs/` (Markdown/MDX).
  - `guides/` = workflows and frameworks
  - `templates/` = copy/paste documents
  - `toolbox/` = checklists and prompts
  - `resources/` = reading list and tools
- The site is built with Astro + Starlight.

## Suggest changes

If you spot an error, want a new topic, or have a template idea, please open a GitHub Issue with:
- the page URL (or file path)
- what you want changed and why
- an example (optional)

You can also submit a PR with edits to the relevant file in `src/content/docs/`.
