title: Prompts & heuristics
---

## Clarifying a vague request
- What user problem are we solving?
- How do we know it exists (evidence)?
- What happens if we do nothing?
- What’s the simplest version that proves value?

## Heuristics
- If it can’t be measured at all, it’s probably not ready to prioritize.
- If there are no non-goals, scope will expand.
